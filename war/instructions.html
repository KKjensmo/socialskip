Instructions

Title:

In this field write a short description about experiment (e.g video title).
WARNING! This field allow only English characters.

VideoURL:

In this field write the url of the Youtube video.


Start & End Time:

Choose a custom start and end video time.
WARNING! If you choose a custom time, interactions will still be stored in the original video time.

Video controls:

Set the time that the user will have to interact with seekbar when pressed Start button.
If you set zero (0), there will be no time limit

Questionarie:

In this field write the url from questionnaire. The questionnaire you can create using Google Drive. Go to drive.google.com. Open page, click the Create button, then Form. Now you can create the form. After creating a form, click on Send form button, copy the link and place this link into the Questionnaire field

Information:

In this field, write the url from instructions document. This document contains information for the user. The instructions you can create using Google Drive. Go to drive.google.com. Open page, click the Create button, then Document. Enter your text and click File and Publish to the web. Click on Start publishing and copy the Document link and place this link into the Information field.

Customize player:

Change the colors of video player and select which buttons to include in the video player.
